The Company

I work on Nodak Insurance Company's three apps: Nodak Insurance, Battle Creek Insurance, and American West Insurance. I continue to develop their iOS and Android apps to this day. Below are some notable features I implemented:

  • Rewrote their Objective-C app to Swift. Also started utilizing MVP architecture on new features.
  • Made all 3 apps use the same code base. iOS uses targets. Android uses gradle product flavors. This made it easier to add features to all 3 apps. (Before was a lot of copy and pasting code.)
  • Created an Apple Watch and Android Watch app. A user can view all their policy information on their watch.
  • Implemented the ability for a user to add their proof of insurance to their Apple Wallet.
  • Redesigned the app and implemented light/dark mode themes.
  • Implemented a social screen pulling in their Twitter feed.(Although we recently removed that.)
  • Added ability for a user to file claims on the apps.
